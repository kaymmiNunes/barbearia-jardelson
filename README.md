# Jardelson Barbearia

Landing page institucional desenvolvida para a **Jardelson Barbearia**, localizada em Remígio-PB. O projeto apresenta os principais serviços, informações de atendimento, localização, redes sociais e contato direto via WhatsApp.

## Objetivo

Criar uma presença digital simples, responsiva e funcional para a barbearia, facilitando o acesso dos clientes a informações como serviços oferecidos, horário de funcionamento, localização e canais de contato.

## Funcionalidades

- Página inicial com chamada para ação.
- Seção institucional sobre a barbearia.
- Lista de serviços oferecidos.
- Área de fotos e vídeos.
- Seção de avaliações.
- Localização com acesso ao mapa.
- Contato direto via WhatsApp.
- Link para o Instagram oficial.
- Layout responsivo para diferentes tamanhos de tela.

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
